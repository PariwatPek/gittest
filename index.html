การกำหนดข้อมูลประเภทงวดการจ่ายเงินเดือน(Payroll)

หน้าจอนี้มีไว้สำหรับกำหนดประเภทงวดการจ่ายเงินเดือน (payroll)




รูปที่1


รูปที่2

รหัส                     : กำหนดรหัสเป็นตัวอักษรภาษาอังกฤษ ไม่เกิน 20 ตัวอักษร
ชื่อไทย                : กำหนดชื่องวดการจ่ายเป็นภาษาไทย
ชื่ออังกฤษ            : กำหนดชื่องวดการจ่ายเป็นภาษาอังกฤษ
จำนวนงวดต่อปี     : กำหนดจำนวนงวดที่จ่ายต่อปีเป็นตัวเลข
ตัวคูณ                  : กำหนดตัวคูณเป็นตัวเลข
ตัวหาร                  : กำหนดตัวหารเป็นตัวเลข
ฐาน                      : กำหนดฐานเป็นตัวเลข


ขั้นตอนการสร้างงวดการจ่ายเงิน
  1. คลิกปุ่ม สร้างงวดการจ่ายเงิน
  2. บันทึกข้อมูลใน fields ต่างๆ
  3. คลิกปุ่ม ยืนยัน เพื่อทำการสร้างงวดการจ่ายเงิน

ขั้นตอนการแก้ไขงวดการจ่ายเงิน
  1. แก้ไขข้อมูลคลิก Icon รูปดินสอ หน้างวดก
  2. ใน fields ต่างๆ
  3. คลิกปุ่ม บันทึก เพื่อทำการแก้ไขงวดการจ่ายเงิน
